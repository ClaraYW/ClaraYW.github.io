Work Plan

Breakdown of My Jewelry Components

I sepparated the whole jewelry box into the following parts for modeling, printing and casting as needed.:

  1. Main Component: A jewelry box frame and necklaces hooks.
  2. Earing Component: A earing baer with ear stud holes and earring hook holders.
  3. Ring component: A ring box with silicon ring cushion.
  4. Bracelet component: A bracelet container
  5. Stretch Goal - Cap Component: A removable cap to close the jewelry box.

Main Component

CAD Modeling

I started by measure my printer and pieces of jewelry to determine the maximum and minimum size of my jewelry box.

Bracelet is something I planded to have it lay flat, so I measured the mininum width of my bracelets.

I used "pipe" and "bend" command to creave the necklace hook.

  • First, I drew a curve.
  • Then, I used "pipe" command to make the root of the curve thick and top of the curve thin.
  • Next, I used "bend" command to bend the surface that was created with the pipe commend.
  • Lastly, I used array to create 7 duplicate hooks.
Necklace Hooks Model

I created a 18x18x9cm box with an 1 cm offset based on the maxinum and minimum range I had measured. I then "booleanUnion" the hooks and the base box.

I cut two sliding slots to the side of the box to allow for the earing component to be installed later. This design also allows a user to easily remove the earing component for cleaning purposes.

I also curve out 4 squares on each side for the cap to snap on.

Earing Bar

CAD Modeling

I first measure the size of my earing studs to determine how big the holes need to be and calculate how many pair of stud holes and earing loops I need to evenly fit in the width of the jewelry box.

I used grasshopper to make the earing stud holes. Originally I started with drawing a circle, then used array, but I want each set of two ear stud holes to be close together and then array the set. Thanks to Junchao, he taught me how to array based on a set of circles and tips to make the model cleaner and more efficient.

  1. I set up 2 points.
  2. I drew two circles using the 2 points as the center.
  3. I used array to produce 6 pair of solid circles on the X axis.
  4. I baked the solid circles.
  5. I substracted these solid circles from the right size rectangle box.
Model in Grasshopper

For the loops at the bottom of the earing bar, I made 1 open cylinder with 1.3mm wall thickness, then used array to duplicate the rest.

I also added two extrusion on each side of the earing bar to allow the bar get slided into the based of the jewelry box.

Ring Component

CAD Modeling

Thoughts behind the design of this component.

  1. Determine the width of the ring box: The design intend is to have half of the bottom for ring and the other half for bracelet.
  2. Determine the height of the ring box: I intend to fill in 60% of the box with silicon ring slots, so that when insert rings inside, there will still be about half of the ring stick outside the silicon slots but all the rings should still be below the full height of the ring box.

Modeled the ring component in Rhino.

  1. I drew a box with the desired height and width.
  2. Removed the cap of the box.
  3. Used "offsetsrf" command to add 3mm thickness to the wall of the box.
  4. I adjusted the offset distance and box width a few times to make sure that
    • It can fix two equal size box inside the main frame.
    • To give is some extra space (0.3mm)to accommodate print tolerance.
Ring and Bracelet Boxes Model

Printing

I started with changing my print bed to glass to see results but end up using the original magnet print bed. It took me long time to adjust bed level and for multiple times to get the better print result. The print time was about 5 hours 38 minutes.

Ring Component with Silicon Slots

Issues:

  1. The original magnetic print bed has dented parts and also unremovable plastics from previous prints. It impacted the appearance of my printouts. I decided to try to use glass bad for this print.
  2. Glass bed had sever adhesion issue.
  3. Changed back to the magnetic bed. the print failed because the wall was too think, it didn't turn into a box. (I was trying to make a thin wall so that there is more room for rings and also I hope it will print faster.)
  4. I adjusted the wall thickness in the model. The second print had corner adhesion issue. (corner will curve upwards)
  5. I then adjusted my bed again and the third print turned out better, but still has slight upward curve at the corner.

Stretch Goal: Cap Component

CAD Modeling

The cap model was built in Rhino.

  1. First, I drewed a square that was 1mm longer and wider than the jewelry box to allow room for print error within tolerance and release some bend presure of the 4 clips.
  2. Then, I extrude the square to be 6mm. (Looking back, I will make it thiner to save some material and time. (e.g., 4mm))
  3. I determined the clip size based the length of the indented part on the main component, how much room I want to leave for user to reach the clip to unclip it.
  4. I left small gaps between the clip and the main box to accommodate print errors withint tolerance.
  5. I then drew the rectangle bar and the triangle, duplicate it 4 times, rotate to the correct side, and combine them with the main cap.

Issue:

    I was having trouble to make the extruded triangle at the right plane. Junchao taught me how to use "Tab" when draw lines to ensure it's drawing the polyline on the same plane.

Fixing & Assembling

Fix the based component with 3D Pen

I thought of using super glue, glue gun, and 3D pen to stick my two parts of the base back together. After some reserach and comparions, I went with super glue plus 3D pen.

Fixing Tools

Process:

  1. Place the bottom of my base on a flat surface.
  2. Apply super glue to the middle part of each side as these are the parts where the top and bottom still touches.
  3. Gentally apply forces and leave it for about 1 minute until I felt the top was not sliding with gental push.
  4. Used my 3D pen with transparent PLA to fill in the caps at the 4 corners.
  5. Used craft knife to remove the extra plastic around the box.

Reflection & Next Step

Reflection

  1. Use cheaper plastic to print and paint the outside: The silk PLA was not cheep, and I felt a bit of ache when I saw lots of the beautiful shinny color was wasted for the infill. I would prabablly use cheaper plaster to print and just color it with nice paint next time when printing big objects.
  2. Adjust the bed more frequently based on the plastic stickiness on the magnet bed to avoid the bed get indented or have plastic permanently stick on them.
  3. Maybe my jewelry box base componenet wall can be thiner to save some print material and time.
  4. Watch out for temperature change in the house to avoide cracks on a big print.
  5. Better account for the thickness and the flexibility of the materials used clips that required bending.

Next Steps

  1. Try the glass bed again with magic glue to see if it sovles the adhesion and upward curing issue with the ring and braclet boxes.
  2. Try out different material and measurement of the clips to fix the cover of the box.

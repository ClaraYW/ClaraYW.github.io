Work Plan

Breakdown My Jewelry Components

I sepparated the whole jewelry box into the following parts for modeling and printing:

  1. Main Component: Jewelry box frame and necklaces hooks.
  2. Earing Bar: Ear stud holes and earring hook holders.
  3. Ring component: 3D printed box with silicon ring cushion.
  4. Bracelet component: 3D printed container
  5. Stretch Goal - Cap Component: A removable cap to close the jewelry box.

Earing Bar

CAD Modeling

I used grasshopper to make the earing holes. Originally I started with drawing a circle, then used array, but I want each set of two ear stud holes to be close together and then array the set. Junchao provided me tips to make the model cleaner and more efficient.

  1. Start with setting up 2 points.
  2. Draw circle using the 2 points as the center.
  3. Used array to produce 6 pair of solid circles on the X axis.
  4. Bake the solid circles.
  5. Substract these solid circles from the right size rectangle box.

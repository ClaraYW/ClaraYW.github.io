Assignment 1: Cookie Cutter

Project Description

Cherry Flower

Image Preparation:

  1. I first find a single cherry flower image online. (I used a real image instead of a clipart so that the shape can be more nature.)
  2. I removed the background in photoshop, and imported to illustrator to convert it into victor image
  3. I exported the victor image as dxf and imported to Rhino.
    Challenges: The stamen in the middle was a bit complex in the original image, and does not turn into victor nicely, so I abandoned the original stamen, and used pen tool to drew some freehand stamens. I wanted to follow the nature feeling, so I made it asymmetrical.

3D Modeling:

  1. I first scaled down the image to a more realistic cookie cutter size
  2. Then I used "join" for the flower petals to make sure all the path are connected.
  3. I used rebuild command to check the total number of points, it was not too many, so I did not make adjustments there.
  4. I then used extrudeCurv command to turn it into polysurface.
  5. I want to allow user to create the stamen pattern on the cookie, so I was thinking of ways to connect the stamen. I thought if connect using thin line between the stamen and the petals, it will impact the look of the cookie with the thin lines.
    1. Idea 1: make the stamen tall so user can use it as a stamp and stamp it in inside the flower mode. Problems:
      1. The stamen have thin middle part, might easily break when pressure applies.
      2. The stamen might not be centered due to the amount of free room inside the flower mode.
    2. To address the two problems above, I thought if I have the stamen attach to a solid flower petals that fits in exactly, then user will always have that stemen centered, as the solid petals will also fit into the hollow cookie cutter.
    3. Then I thought about user will need a handle to stamp it like a stamp after you press the flower ring, so I added a cylinder.
    4. I wanted to make the cylinder easier to grab, so I indented two side with 2 boxes toward the top of the cylinder.
    5. To make the stamen stamp, I attached the stamen and solid flower petals and the handle together.

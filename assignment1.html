Assignment 1: Cookie Cutter

Project Description

Cherry Flower

Image Preparation:

  1. I first find a single cherry flower image online. (I used a real image instead of a clipart so that the shape can be more nature.)
  2. I removed the background in photoshop, and imported to illustrator to convert it into victor image
  3. I exported the victor image as dxf and imported to Rhino.

Challenges: The stamen in the middle was a bit complex in the original image, and does not turn into victor nicely, so I abandoned the original stamen, and used pen tool to drew some freehand stamen. I wanted to follow the nature feeling, so I made it asymmetrical.

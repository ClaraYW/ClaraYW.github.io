Final Project Proposal

Concept

Background

I have a bunch of jewelry that I have been strugging to find a way to better organize them. There are lots of jewelry organizer you can buy online, but they are either not exactly the type I want. (e.g., have all the spots for all types of jewelry I have), or the ones I like are too expensive and I couldn't justify the cost. So I have been just keeping all of them in a bag, which is not ideal.

Problems:

  1. My necklaces always tangle together when kept all in one bag.
  2. To avoide things tangle together, I sometimes don't want to put them back in the bag after finish using it. This cause me not able to find my jewelries when next time want to use them.
  3. Putting them all separated in different bags is troublesome, so I end up not Putting them back in the bag again.

Requirements:

  1. An organizer that has a custom spot for all different type of jewelries I have.
  2. I can move it around easily.
  3. It will stay in place when I put them in a location.
  4. It's easy for me to take and put my jewelries back.

Sketches & Rendering

Breakdown of Tasks

CAD Modeling, 3d printing, finishing

  1. Measure and determine the size of the jewelry organizer I want, every piece also needs to be be within the constrain of the printer.
  2. Grasshopper: Draw shapes that requires size adjustments using Grasshopper. (e.g.,like the size of the ring holder, bracelet holder, etc.)
  3. Rhino: Draw the main container, combine the different jewerly holder polysurfaces, produce the render view, export STL file(s) in Rhino.
  4. Cura: Slice objects in Cura. (I might need to break it into few pieces and find ways to connect them later)
  5. 3d Printing: Print the object(s) using ender 3 pro.
  6. Finishing: Clean up any extra filaments, polish the surface.

Fix Naked Edge of a Mesh

I downloaded the smiley minion file from Thingiverse. The original file has two separated parts, the front and back of the minion.

Smily Minion Original STL

First, I needed to clean the mesh before merging them. I noticed the 2 meshes were closed mesh, however, when run Mesh Repair, it says it's bad mesh. I hit repair button, and also cicked around some other controls. I end up break the mesh and caused 1 naked edge occured.

I then messed around for a very long time but couldn't fix it effectively, SOS Junchao and he troubleshoot with me and we tried different methods to finally fixed it. Below are the steps we took.

  1. Use command + sift to select around the broken surface and delete the broken meshes
  2. Export the object so that all the surfaces are broken into separate pieces
  3. Select the broken piece and use invert command to insert select all the other pieces
  4. Hide the selected good pieces
  5. Use polyline to connect the mesh points (turn on Vertex, it helps to get the cross snap to the exact mesh point)
  6. Select two meshline and user the command loft to create a surface
  7. If step 7 didn't work as expected, convert mesh into Nurb (MeshtoNurb) then repeat step 7. (This helped to fix the less complex mesh lines where you can see the limited lines pretty clearly.)
  8. To fix the more complex mesh lines where there are so many lines that looks overlapping. Use "Cap" to cap all empty surfaces
  9. Then convert the Nurb back to mesh, and you can adjust how many surface you want to have.

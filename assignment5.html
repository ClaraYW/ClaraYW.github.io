Make Lamp

Design & Sketch Ideas

I changed my last week's idea from doing to box lamp to a shell lamp, and make the light bulb like a pearl in the shell. I also want to add patterns to the shell so that when the light is on, it can be reflected on the wall.

About the lamp: there were a few features I thought about:

  1. Have a support at the base of the lamp that help to lift it up so the light bulb doesn't touch the plastic bottom directly.
  2. Have a connector that connects the top and bottom shell so that when lift the other side it will open like a shell.
    • User should be able to change the light bulb easily with one side can be open as wide as needed.
    • The connector needs to be removable so that user can easily remove the socket.
    • The connector is strong enough to keep the top and bottom shell together on one side. Lifting the other side up should not disconnect the connected side easily.
  3. Have a mini stepper on the top shell and a stick at the bottom shell, so that user can adjust different angle of the shell opening and lock the angle with the stick and stepper.

About the connector/clip: Originally, I thought about using a clip that has a handle so user can easily open it up and clip the top and bottom of the shell. Then I was inspired by the zip ties and put down the below requirements for my connector/clip

  • The clip needs to be flexible so that it can be easily opened without breaking.
  • Instead of a handle which might ruin the aesthetics of the perfectly oval shapped Lamp, I decided that it just needs a way for user to open it easily, so if it's a ring shape and it's big enough to fit two fingers to easiy expand it, that works too.
  • The ring cannot be too big that the connection between the shells become loose.

Turn it into mode of a mode

I started with how I want the final product look like. Then made a mode for the final product (what the silicon mode should look like). Lastly, flip the extrusion to make the mode of the silicon mode.

Details steps:

  • Draft Analysis: to adjust the top and bottom angle, and set the parting line. Issue: The added draft curve didn't get separated during explode.
  • Explode: the object so I can separate the object into two pieces. (Didn't work well for me cause it explode with curves that are perpendicular to each other.
  • BooleanDifference: to separate the original object insteaad and created the parting line in the middle of the heart.
  • Removed the cap of the cutted side of the surface.
  • DupeEdge to allow selection of the curves on the cutted side of the surface.
  • RibbonOffset: Select curves then ribbonoffset (which I later realize that I might not need this because of my parting line is already all at the same level on the z-axis.) I also realized the default RibbonOffset is not perfectly following the parting by default. So I used "Add slash" to address it.
  • Offset curve: twice of the outside ribbon curve.
  • PlanarSurf: to turn the area betwen two curves into a surface
  • BlenderSrf: to bridge between two surfaces
  • Adjust z-axis to be the same if needed. I didn't need this step.
  • Extrude Curve to the correct direction.
